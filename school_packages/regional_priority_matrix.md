# Regional Priority Matrix

This matrix translates Wen Quan's stated filters into application priorities.

## User Filters

- Main goal 1: save money.
- Main goal 2: choose PhD routes that can support local immigration/PR after graduation.
- If a PhD route has roughly high PR feasibility after graduation, self-funded study may be considered.
- If a PhD route is likely funded, it may be considered even when immigration is less direct.
- If a school is not highly ranked, not likely to fund, and not helpful for immigration, do not prioritize it.

## Current Regional Priority

| Priority | Region | Reason |
|---|---|---|
| S | New Zealand | Low PhD tuition for international doctoral students, strong post-study work route, skilled migration can value PhD-level qualifications, but job offer still matters. |
| S | Canada Ontario | Funded PhD common in CS; Ontario has a PhD graduate immigration stream that can be strong for doctoral graduates. |
| S | Hong Kong | HKPFS and university studentships can make PhD financially attractive; IANG and 7-year ordinary residence route are clear. |
| A | Sweden / Finland / Norway / Germany / Netherlands | Best when the PhD is a paid doctoral employment or funded project; immigration usually depends on continued work after graduation. |
| A- | Australia | Strong universities and RTP scholarships, but self-funded PhD is expensive and PR is points/job/occupation dependent. |
| B+ | United States | Excellent if fully funded and if later NIW/EB route is viable; self-funded CS PhD is not recommended. |
| B | United Kingdom | Good only with strong university/funding; Graduate Route alone is not a robust PR path. |

## Default Application Strategy

1. Start with New Zealand, Ontario Canada, and Hong Kong.
2. Add Europe only when there is a paid doctoral position or strong supervisor match.
3. Add Australia only for RTP/scholarship or a top supervisor.
4. Add the US only for fully funded PhD programs or labs strongly aligned with AI/HCI/accessibility.
5. Avoid unfunded, low-ranked, weak-immigration options.
