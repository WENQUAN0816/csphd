# School-Specific PhD Application Packages

This directory follows the project rule: one target school, one application package.

Each school package is a working folder for tailoring CV, research concept note, research proposal, portfolio statement, and outreach emails to that school's supervisors and funding/immigration context.

## Regions

- `canada/`
- `new_zealand/`
- `australia/`
- `hong_kong/`
- `usa/`
- `uk/`
- `europe/`

## Use

Start from the generic XMUM materials in `application_materials/`, then adapt the content inside each school package before sending.

Do not send the generic XMUM version unchanged to other universities.

## Current Coverage

The first-wave package set contains 43 school packages:

- Australia: 5
- Canada: 6
- Europe: 10
- Hong Kong: 5
- New Zealand: 5
- United Kingdom: 5
- United States: 7

These are first-draft tailoring packages. They are suitable for planning, prioritization, and drafting. Before real outreach, supervisor names, emails, current affiliations, funding deadlines, and official immigration rules must be verified.

## Important Warning

Do not bulk-send these emails. Each `outreach_email.md` still requires:

- one verified supervisor name;
- one specific recent paper or project;
- one accurate funding/program pathway;
- school-specific attachments from that school's `attachments/` directory.
