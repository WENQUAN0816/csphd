# 04 技术栈与学习路线

## 总体判断

这个方向的核心不是会一个大模型，而是把室内空间转化为 AI 可以理解、推理、预测和生成的结构。

应优先掌握：

1. Python + PyTorch
2. Computer Vision
3. 3D Vision
4. VLM/LLM
5. Scene Graph
6. Generative Layout Design
7. Interior accessibility and age-friendly evaluation

强化学习、机器人控制和大规模世界模型训练可以后置，不适合作为入门阶段重点。

## AI 基础

| 模块 | 需要掌握什么 | 作用 |
|---|---|---|
| Python + PyTorch | 数据处理、模型训练、实验复现 | 基本门槛 |
| 机器学习基础 | classification, regression, clustering, metrics | 空间风险评估、环境质量预测 |
| 深度学习 | CNN, Transformer, attention, embedding | 图像、文本、空间特征建模 |
| Computer Vision | detection, segmentation, depth estimation | 识别家具、门窗、障碍物 |
| 3D Vision | point cloud, mesh, voxel, NeRF, 3DGS | 从照片/扫描重建三维室内空间 |
| VLM/LLM | CLIP, BLIP, LLaVA, GPT-style reasoning | 让模型解释空间是否适合老人 |
| Generative Models | diffusion, layout generation, scene generation | 自动生成室内改造方案 |
| Graph / Scene Graph | nodes, edges, spatial relations, GNN | 表达床、门、通道、家具关系 |
| Evaluation | mAP, IoU, F1, human evaluation | 证明模型有效 |

## 空间智能核心技术

### 1. 空间重建

目标：把照片、视频、全景图、LiDAR、点云、平面图变成 3D 空间。

技术：

- SLAM
- Structure from Motion
- Depth Estimation
- Point Cloud
- 3D Gaussian Splatting
- NeRF
- BIM/IFC reconstruction

### 2. 空间理解

目标：让 AI 知道空间里有什么、在哪里、关系是什么。

技术：

- Object detection
- Semantic segmentation
- Instance segmentation
- 3D object detection
- Scene graph generation
- Spatial relation reasoning

### 3. 空间评估

目标：让 AI 判断空间好不好。

评价对象：

- 无障碍性
- 跌倒风险
- 夜间行动安全
- 家具拥挤度
- 通行效率
- 护理便利性
- 光环境、热舒适、声环境、空气质量

技术：

- Multimodal classification
- Risk scoring
- Rule-based + AI hybrid evaluation
- Explainable AI

### 4. 空间生成

目标：让 AI 自动提出新的布局和改造方案。

技术：

- Layout generation
- Diffusion models
- Constraint optimization
- Graph generation
- Text-to-3D scene

### 5. 空间编辑

目标：不是从零生成，而是在现有户型上修改。

技术：

- Instruction-based editing
- VLM-guided design editing
- Constraint-aware generation

### 6. 空间仿真

目标：在虚拟环境中测试人或机器人如何行动。

技术：

- Agent simulation
- Human behavior simulation
- Navigation
- Affordance learning
- Digital twin

## 室内环境基础

| 室内环境模块 | 需要研究什么 | 如何和 AI 结合 |
|---|---|---|
| 人体工学 | 尺寸、可达范围、坐站转换、轮椅尺度 | 转化为空间约束和评价规则 |
| 老龄友好设计 | 跌倒预防、认知友好、照明、扶手 | 转化为 risk score |
| 室内空间布局 | 动线、功能分区、家具关系 | 用 scene graph 和 layout model 表达 |
| 建筑规范 | 无障碍规范、消防、通行宽度 | rule-based evaluation |
| 环境舒适 | 光、热、声、空气质量 | 传感器数据和预测模型 |
| 行为研究 | 老人日常活动、护理路径 | agent simulation 或 activity recognition |
| POE 评价 | Post-Occupancy Evaluation | AI-assisted environmental assessment |
| BIM / CAD | 平面图、IFC、Revit、SketchUp | 与 AI 的 3D/语义数据连接 |
| 空间句法 | connectivity, visibility, integration | 空间结构特征 |
| 循证设计 | design evidence, health outcomes | 支撑理论框架 |

## 推荐学习顺序

### 第 1 阶段：3 个月

- Python 数据处理
- PyTorch 基础
- YOLO / SAM / GroundingDINO / CLIP 复现
- 室内图片中的家具检测和分割
- 建立小型标注集

### 第 2 阶段：6 个月

- Depth estimation
- Point cloud 基础
- ScanNet / Matterport3D / Structured3D 数据处理
- 房间级老龄友好风险评分模型
- VLM 对空间风险的解释

### 第 3 阶段：12 个月

- Scene graph
- 3D scene understanding
- 室内空间可达性和跌倒风险评价模型
- 专家标注数据集
- 第一篇方法论文或数据集论文

### 第 4 阶段：2-3 年

- Generative layout design
- Counterfactual spatial editing
- Human-centered simulation
- 室内空间世界模型原型系统
- 完整博士论文
