# 02 学费预算

预算按国际生、全日制博士、只算学费和必要学校费用、不含住宿生活费估算。汇率按 2026-06-10 附近估算：

- RM1 约等于人民币 1.67 元
- USD1 约等于 RM4.06

## 预算判断

低预算优先级：

1. USM
2. XMUM
3. UM

高品牌/强资源优先级：

1. Monash Malaysia
2. UM
3. Nottingham Malaysia

现实判断：

- 如果需要控制学费，USM、XMUM、UM 更合适。
- 如果追求学校品牌、国际认可度和 AI/CS 导师资源，Monash 和 Nottingham 更强，但成本高很多。
- 已经有博士后，申请第二博士奖学金的难度较高，尤其 Monash 的部分研究生奖学金对同层级第二博士不友好。
