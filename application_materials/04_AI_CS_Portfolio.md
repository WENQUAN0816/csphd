## Technical Features

- Single-page web application using HTML, CSS, and JavaScript.
- Interactive map interface using Leaflet.
- Radar chart visualization using ECharts.
- Multidimensional evaluation scoring system.
- Bilingual interface, now set to English by default for international presentation.
- Community-level data visualization with score categories, filters, and detail panels.
- Scoring dimensions include physical space, health services, transport, social participation, safety and emergency, daily services, mental wellness, and governance.

## Proposed Portfolio Extension for PhD Application

The next version can extend the current demo into:

1. **Indoor Space Risk Assessment Module**
   - Input: indoor image, simple room description, or floor-plan screenshot.
   - Output: age-friendly risk categories and preliminary risk score.

2. **Knowledge-Guided Recommendation Module**
   - Uses an age-friendly design knowledge base.
   - Generates design recommendations such as improving lighting, adding handrails, clearing circulation paths, or modifying furniture layout.

3. **Expert Validation Interface**
   - Allows experts to rate whether AI-identified risks and recommendations are correct, useful, and explainable.

4. **Comparison Mode**
   - Compares generic VLM output with knowledge-guided output.
   - Measures improvement in relevance, consistency, and explanation quality.
