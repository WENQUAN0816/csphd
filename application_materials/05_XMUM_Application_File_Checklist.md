## Notes

- Do not push passport, transcripts, certificates, or ID photos to a public GitHub repository.
- If `csphd` is created as a private repository, still consider keeping sensitive identity documents out of git and uploading them directly to XMUM's official application portal.
- Check whether XMUM requires certified true copies, notarized translations, or original English versions.
